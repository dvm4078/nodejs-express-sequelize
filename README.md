## Required

1: Nodejs v12.16.0 (latest) <br />
2: Yarn (https://classic.yarnpkg.com/en/docs/install#mac-stable) <br />

## Deploy prod

1: Install dependences

```bash
yarn
```

2: Build

```bash
yarn build
```

3: Start

  3.1: Start main service
  ```bash
  yarn start-main
  ```

  3.2: Start websocket service
  ```bash
  yarn start-socket
  ```

  3.3: Quick start
  ```bash
  yarn start
  ```

### Dev local

1: Install dependences

```bash
yarn
```

2: Start

```bash
yarn dev
```
